🌸 關於幽幽子 Bot

💻 技術架構

  • ▸ 開發語言：Python 3.10+
  • ▸ 核心框架：Py-cord v2.6.1 (Discord.py 分支)
  • ▸ 數據管理：SQLite + JSON + YAML
  • ▸ 非同步處理：asyncio + aiohttp
  • ▸ 數據視覺化：Matplotlib

🚀 部署與運行

  • ▸ 運行環境：Linux / Docker / PM2
  • ▸ 啟動方式：Systemd Service / Docker Compose
  • ▸ 日誌系統：雙層日誌 (main-error.log + commands_error.log)
  • ▸ 自動備份：每日凌晨 4 點 SQLite 熱備份
  • ▸ 錯誤追蹤：企業級 commands_error_logger
