**Yuyuko Bot 官方宣傳網站** —— 以東方Project 西行寺幽幽子為主題的精美靜態網站。

🌸 冥界櫻花與玻璃擬態風格的華麗 Bot 介紹頁。

## ✨ 特色

- 即時櫻花粒子飄落特效
- 細膩玻璃擬態（Glassmorphism）設計
- 完整指令、職業系統、技術架構展示
- 完美響應式，行動裝置體驗佳
- Tailwind CSS + 原生 JavaScript

## 📄 網站頁面

- **首頁** (`index.html`) — Bot 介紹與一鍵邀請
- **指令介紹** (`pages/commands.html`) — 50+ 個指令詳細說明與範例
- **職業系統** (`pages/jobs.html`) — 正副職介紹（含薪資、被動、彩蛋）
- **技術架構** (`pages/tech.html`) — 後端技術棧與部署說明

## 🚀 部署方式（GitHub Pages 推薦）

1. Fork 本專案
2. 進入 **Settings → Pages**
3. 設定 Source 為 `Deploy from a branch` → `main` 分支 → `/ (root)`
4. 儲存後等待部署完成

部署後可透過 `https://你的帳號.github.io/Yuyuko-Web` 訪問。
