**Official Website for Yuyuko Bot** — A beautiful static website themed around Touhou Project's Yuyuko Saigyouji (西行寺幽幽子).

🌸 Filled with elegant Netherworld sakura and glassmorphism design.

</div>

## ✨ Features

- Real-time falling sakura particle animation
- Stunning Glassmorphism + Touhou aesthetic
- Complete command list, job system, and technical architecture pages
- Fully responsive design (excellent on mobile)
- Built with Tailwind CSS + Vanilla JavaScript

## 📄 Pages

- **Home** (`index.html`) — Bot introduction, features, and invite button
- **Commands** (`pages/commands.html`) — Detailed explanations for 50+ slash commands
- **Jobs** (`pages/jobs.html`) — Main and secondary job system (with salary, skills, and fun easter eggs)
- **Tech Stack** (`pages/tech.html`) — Backend technologies and deployment guide

## 🚀 Quick Deployment (GitHub Pages)

1. Fork this repository
2. Go to **Settings → Pages**
3. Set Source to `Deploy from a branch` → `main` → `/ (root)`
4. Save and wait for deployment

Your site will be live at: `https://yourusername.github.io/Yuyuko-Web`
